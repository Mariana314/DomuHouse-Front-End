# DomuHouse Frontend

## About the Project

DomuHouse is a real estate web application that allows users to browse, publish, and manage properties. The frontend provides an interactive and responsive user interface with role-based views for clients, agents, and administrators.

This project was developed collaboratively by a team of 5 developers.

---

## My Contributions  

- Designed the user interface and overall visual structure of the web application.  
- Contributed to frontend development using React, implementing key components and views.  
- Participated in backend development by creating routes and handling server-side logic.  
- Integrated frontend and backend, ensuring proper communication between the client and the API.

---

## Features

* Responsive user interface
* Property listing and detailed views
* Advanced property search and filters
* Role-based dashboards (Client, Agent, Admin)
